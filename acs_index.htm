1. 동영상 제작(다음은 참고 자료입니다)

    • 수업 영상 제작 방법: 파워포인트 슬라이드 쇼 녹화
    • 수업 영상 제작 방법: 파워포인트 화면 녹화
    • 수업 영상 제작 방법: (갤럭시탭(android)) 화면 녹화
    • 수업 영상 제작 방법: (아이패드(ios)) 화면 녹화
    • (android / ios) VLLO(앱)를 이용한 영상 편집

  2. 이곳을 클릭하여

    1. 적합한 제목 생성
    2. 동영상 업로드

  3. 이곳을 클릭하여

    1. (양식) 복제 버튼 클릭
    2. 양식 제목을 변경
    3. 1번 문항에 동영상 삽입
    4. 응답 수집 링크와 공동 작업 링크 생성

  4. 이곳을 클릭하여 응답 수집 링크와 공동 작업 링크 전달
